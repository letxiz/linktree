# Linktree | Letícia Alves

Projeto de página pessoal no estilo Linktree, feito com HTML e CSS. A página reúne links para GitHub, LinkedIn, projetos, contato por e-mail e uma página com informações sobre mim.

## Sobre o projeto

O site possui duas páginas principais:

- `index.html`: página inicial com foto, descrição curta, redes sociais e links principais.
- `sobre.html`: página com uma apresentação mais completa e botão para voltar à página inicial.

O visual usa um card centralizado com fundo em gradiente, efeito de transparência, botões de navegação e ícones do Font Awesome.

## Tecnologias usadas

- HTML5
- CSS3
- Font Awesome

## Estrutura de arquivos

```text
linktree/
├── assets/
│   ├── leticia.jpg
│   └── styles.css
├── index.html
├── sobre.html
└── README.md
```

## Como executar

Como o projeto é estático, não precisa instalar dependências.

1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` no navegador.

Também é possível usar a extensão Live Server no VS Code para visualizar o projeto com recarregamento automático.

## Funcionalidades

- Links para GitHub e LinkedIn.
- Botão para acessar os repositórios no GitHub.
- Página "Sobre mim".
- Link direto para contato por e-mail.
- Layout responsivo para telas menores.

## Personalização

Para alterar informações do perfil, edite os textos e links em `index.html` e `sobre.html`.

Para alterar cores, espaçamentos, botões e responsividade, edite o arquivo `assets/styles.css`.

Para trocar a foto de perfil, substitua o arquivo `assets/leticia.jpg` ou atualize o caminho da imagem no HTML.
